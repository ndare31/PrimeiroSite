    Aula 6 - Atributos das tags

    (Voltar)

    Atributos de tags são propriedades que as tags podem ter. Essas são algumas dos atributos mais genéricos.

    • Id é um atributo que serve para identificar um elemento, isso é importante quando se trabalha com outras linguagens em conjunto, como CSS ou JS.
    • Style é um atributo que permite que usemos o CSS em um código HTML, ele é mais focado para o CSS, mas pode ser utilizado no HTML ou JS.
    • Class é outro atributo que permite padronizar algum elemento, como um título ou texto. Iremos focar mais no curso de CSS

      • Alguns atributos são mais específicos para algumas tags, como a tag style que usam o atributo type para definir se seu valor será numérico, texto ou cor. A tag input cria uma caixa.

      A tag img pede um atributo específico chamado src que endereça a imagem, identificando se ela vem do próprio computador ou da internet. Além de atributos como widht que permite modificar o comprimento da imagem.

    Aula 7 - Textos

    (Voltar)
    1. A tag h significa header, cabeçalho em português, é numerado de 1 a 6, sendo o 1 o maior e o 6 menor.
    2. A tag p é responsável por criar parágrafos, textos.
    3. A tag blockquote cria uma citação no corpo do texto.
    4. A tag strong cria um texto em negrito.
    5. A tag i cria um texto em itálico.
    6. A tag u significa underline, ou seja, sublinhado.
    7. A tag sup é uma forma digitar um texto menor na parte de cima de um texto normal, como em marcas registradas.

    Aula 8 - Listas Ordenadas e Não Ordenadas

    (Voltar)

    A tag ol é uma tag pai, ou seja, ordinary list, que define o tipo de lista e necessita de tags filhas, como a li que significa list-item. No caso das Ordenadas elas são numeradas, mas podem ser modificadas no CSS. Já na ul, unordinary list, que também é uma tag pai, ela não vai numerar os itens.

    Aula 9 - Links

    (Voltar)

    A tag que linka sites, é a e ela necessita de um atributo para conseguir fazer isso, que é chamado href, serve para referenciar um link. Exemplo:

    Site da DIO
